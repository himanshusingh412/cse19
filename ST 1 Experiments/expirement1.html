Course Description

Data Structures and Algorithms (DSA)
This course covers fundamental data structures such as arrays, linked lists, stacks, queues, trees, and graphs, along with algorithms for searching, sorting, and traversing these structures.
Artificial Intelligence (AI)
This course introduces the basic concepts of AI, including ML, neural networks, natural language processing, and computer vision.
